# Rally Encuentro de Capitanes

Prototipo web para evaluar participantes del Rally Encuentro de Capitanes.

## Como abrirlo

Abre `index.html` en un navegador moderno.

## Credenciales de prueba

Administrador:

- Usuario: `admin`
- Contrasena: `admin2026`

Evaluadores:

- `eval.tierra1` / `tierra1`
- `eval.tierra2` / `tierra2`
- `eval.agua1` / `agua1`
- `eval.agua2` / `agua2`
- `eval.aire1` / `aire1`
- `eval.aire2` / `aire2`
- `eval.fuego1` / `fuego1`
- `eval.fuego2` / `fuego2`

## Datos de simulacion

La version actual incluye:

- 50 participantes ficticios.
- 8 equipos.
- 8 estaciones: Hippos, Llenala, Palos samuray, Mision espuma, Torre infinita, Alfombra magica, Vasos al aire y Deslizate.
- La mitad de los participantes ya evaluados.
- La otra mitad pendiente para demostrar el flujo de evaluacion.

## Nota de seguridad

Esta version es un prototipo estatico. No debe usarse en produccion con datos sensibles reales sin backend, autenticacion segura y base de datos.
